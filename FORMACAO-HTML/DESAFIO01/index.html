title

(Voltar ao início)

O tittle é o que aparece na aba do usuário.

Um exemplo visual da tag tittle.

Exemplo em código: <title> Nome do Tittle </title>

h1,h2,h3,h4,h5,h6

(Voltar ao início)

Essas tags executam a mesma função, que seria destacar um tema/titúlo. A diferença entre elas é que cada uma apresenta um tamanho diferente.

Um exemplo visual das tags h1 a h6:

  1. h1

  2. h2

  3. h3

  4. h4

  5. h5
  6. h6

Exemplo em código: <h1> Nome de um tema </h1>

p

(Voltar ao início)

A tag p seria basicamente o paragráfo do HTML. Onde seria inserido os blocos de textos/strings.

Um exemplo visual da tag p. Que seria a maior parte das strings que o usuário visualiza.

Exemplo em código: <p>Bloco de texto</p>

mark

(Voltar ao início)

Mark é uma tag usada dentro de alguma tag que faz com que uma string , seja marcada. Similar a um marca texto na vida real.

Um exemplo: String sem mark VS String com mark.

Exemplo em código: <p<<mark>Bloco de texto</mark></p>

small

(Voltar ao início)

Small é uma tag usada para diminuir o tamanho de um texto.

Um exemplo: Texto Normal VS Texto com small.

Exemplo em código: <small>Bloco de texto</small>

i

(Voltar ao início)

O i é uma tag utilizada para formatar strings em fonte itálica.

Um exemplo: String sem i VS String com i.

Exemplo em código: <p><i>Bloco de texto</i> </p>

u

(Voltar ao início)

A tag u é usada para formatar um underline dentro de uma string.

String sem u VS String com u

Exemplo em código: <p><u>Bloco de texto</u></p>

strong

(Voltar ao início)

A tag strong é usada para formatar a string em negrito.

Um exemplo: String sem strong VS String com strong.

Exemplo em código: <p><strong>Bloco de texto</strong></p>

ol

(Voltar ao início)

A tag ol é usada para fazer uma lista de forma ordenada.

ul

(Voltar ao início)

A tag ul é usada para fazer uma lista de forma desordenada.

li

(Voltar ao início)

A tag li é usada dentro de ol ou ul, para listar a lista.

ol VS ul

Como ol e ul tem o mesmo intuito, mas fazem de formas diferentes. Seria fundamental detalhar a diferença e mostrar o exemplo lado a lado de ambas.

  1. Isso
  2. é
  3. um
  4. exemplo
  5. de
  6. lista
  7. ordenada(ol).
VS
  • Isso
  • é
  • um
  • exemplo
  • de
  • lista
  • desordenada(ul).

Exemplo em código de ol: <ol><li>Bloco de texto</li></ol> VS Exemplo em código de ul: <ul><li>Bloco de texto</li></ul>

a

(Voltar ao início)

A tag a é usada para linkar determinada funcionalidade dentro de uma página HTML, com intuito de redirecionar o usuário ao clicar na funcionalidad clicável a outra página, ou outra aba, ou outro site, ou até mesmo redirecionar dentro da própria página.

Isso é um exemplo de string sem o uso da tag a.Esse é o exemplo de uma string com uso da tag a. Clique aqui para testar e entender o exemplo.

Exemplo em código: <p><a href="link a ser inserido">Bloco de texto</a></p>

Exemplo para tag a.

Voltar para tag a

hr

(Voltar ao início)

A tag hr é utilizada para traçar uma linha dentro da página do HTML, diante e onde do que o desenvolvedor ache melhor para utilizá-la em seu código.

Acima é apresentado uma representação visual do que é a tag hr. Essa linha acima dos temas é um hr.

Exemplo em código: <hr />

sub

(Voltar ao início)

A tag sub é usada para formatar determinada string um pouco abaixo da altura de sua linha.

Um exemplo: String sem sub VS String com sub.

Exemplo em código: <p><sub>Bloco de texto</sub></p>

sup

(Voltar ao início)

A tag sup é usada para formatar determinada string um pouco acima da altura de sua linha. O contrário de sub.

Um exemplo: String sem sup VS String com sup.

Exemplo em código: <p><sup">Bloco de texto</sup>/p>

blockquote

(Voltar ao início)

A tag blockquote no HTML é usada para citar um bloco de texto de outra fonte, como uma citação de um livro ou um trecho de um artigo.

Isso é um blockquote. Aqui poderia ser usado para citar uma referência ou citação. Mas não é o caso agora.

Exemplo em código: <blockquote>Bloco de texto</blockquote>

font

(Voltar ao início)

A tag font se tornou absoleta com a nova versão do HTML, antes era uasada para definir a fonte da string. Hoje são usados comandos do css para definir a font.

Isso aqui é um bloco de string com uma fonte.

VS

Isso aqui é outro bloco de string com uma fonte diferente.

Exemplo em código: <p style="tipo da fonte">Bloco de texto</p>

del

(Voltar ao início)

A tag del é usada para representar texto que foi deletado ou removido de uma versão anterior de um documento. Ela é frequentemente usada em conjunto com a tag ins (insert) para mostrar alterações entre versões de documentos.

String com del ins

Exemplo em código: <p><del>Bloco de texto</del><ins>Bloco de texto novo</ins></p>

abbr

(Voltar ao início)

A tag abbr é usada para definir uma abreviação. Essa tag ajuda a fornecer uma explicação ou expansão completa do termo abreviado ou do acrônimo, que pode ser exibido quando o usuário passa o mouse sobre o texto.

Isso é um texto do HTML sem o uso do abbr. Isso é um texto do HTML com o uso de abbr.

Para entender melhor a diferença passe o mouse acima do HTML.

Exemplo em código: <p><abbr tittle="titulo da abreviação">abreviação</abbr>/p>
